# Travel Itinerary App ✈️

A full-stack planning application where users can create trips and organize destination with time-based itineraries

---

## Features

- Choose country
- Create trips
- Add planed items with the following category (activities, meals, accommodation)
- Time-based itinerary sorting
- Cost tracking
- Google Maps integration support

---

## 🧱 Tech Stack

### Backend

- Java
- JDBC
- PostgreSQL

### Frontend

- HTML
- CSS
- Javascript

## 🗂️ Data Model

PlannedItem is the core entity:
- type (activity / meal / accommodation)
- dateTimeStart
- dateTimeEnd (optional)
- cost (optional)

See '/docs' for full design
